# My List - TypeScript Project

[Click Here for Live Demo](https://comfy-beignet-79e166.netlify.app/)

## Description

This TypeScript project, "My List," is a basic to-do list application. It allows you to add, remove, and mark items as done.

## Features

- Add new items to the list by entering text and clicking the "+" button.
- Remove items from the list by clicking the "X" button.
- Mark items as done by checking the checkboxes.
- Clear the entire list with the "Clear" button.

Enjoy using "My List" to keep track of your tasks!
